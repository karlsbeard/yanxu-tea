## 🚀 快速开始 | Quick Start

### 安装依赖 | Install Dependencies

```bash
npm install
```

### 开发模式 | Development

```bash
npm run dev
```

访问 <http://localhost:5173> 查看网站

### 构建项目 | Build

```bash
npm run build
```

## 🛠️ 技术栈 | Tech Stack

- **Frontend**: React 19.1.0 + Vite 6.0.7
- **Styling**: Tailwind CSS 3.4.14
- **Routing**: React Router 6.28.0
- **i18n**: React i18next 15.1.0
- **Icons**: Lucide React
- **Deployment**: GitHub Pages
